Data from IMAGe Project prepared to use with IMAGe Studio software (www.gpem.uq.edu.au/qcpr-image)
--------------------------------------------------------------------------------------------------
Internal migration data of Sudan
Based on migration and population data from: IPUMS (https://international.ipums.org/international)
Year: 2008
Number and original administrative geography of BSUs: 25 States
Migration data type: Transition
Length of migration interval in years: 1
Contiguity file based on digitial boundary data from: GADM (http://www.gadm.org/)
