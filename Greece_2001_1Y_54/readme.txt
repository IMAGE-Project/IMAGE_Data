Data from IMAGe Project prepared to use with IMAGe Studio software (www.gpem.uq.edu.au/qcpr-image)
--------------------------------------------------------------------------------------------------
Internal migration data of Greece
Based on migration and population data from: IPUMS (https://international.ipums.org/international)
Year: 2001
Number and original administrative geography of BSUs: 54 Prefectures
Migration data type: Transition
Length of migration interval in years: 5
Contiguity file based on digitial boundary data from: GAUL (http://www.fao.org/)
