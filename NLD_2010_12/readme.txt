Data from IMAGe Project prepared to use with IMAGe Studio software (www.gpem.uq.edu.au/qcpr-image)
--------------------------------------------------------------------------------------------------
Internal migration data of The Netherlands
Based on migration and population data from: National Statistical Agency (www.cbs.nl)
Collection instrument: Register
Year: 2010
Number and original administrative geography of BSUs: 12 Provinces
Migration data type: Event
Length of migration interval in years: 1
Contiguity file based on digitial boundary data from: NSA (www.cbs.nl)
